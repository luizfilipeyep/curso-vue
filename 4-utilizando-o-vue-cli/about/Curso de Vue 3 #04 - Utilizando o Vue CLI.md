# Curso de Vue 3: #04 - Utilizando o Vue CLI

Neste vídeo você vai aprender a instalar o [#Vue](https://www.youtube.com/hashtag/vue) em um projeto pelo CLI e também a configurar toda a aplicação pelo cliente. Utilizando Vue Router, Babel e outras opções disponíveis.

------



- O **CLI** é uma ferramenta de linha de comando do Vue;
- Possibilita **criar e configurar** projetos de forma mais avançada;
- **Por exemplo:** adicionar o Vue Router desde a instalação do projeto;
- Isso flexibiliza muito as coisas, vamos então instalar o Vue CLI e testar ele!



## Instalação do CLI

Para instala-lo vamos utilizar o npm que vem com o node.js



## Criando Projeto

Basta rodar no terminal este comando, e executar as configurações necessárias:

~~~
vue create {nome-do-projeto}
~~~



## Abrindo o Projeto

Para abrir, criamos um servidor local:

~~~
npm run serve
~~~
